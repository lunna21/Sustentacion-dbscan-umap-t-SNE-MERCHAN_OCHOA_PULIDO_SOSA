## Objetivo del Proyecto:
Integrar los conocimientos teóricos y metodológicos expuestos en las presentaciones grupales del cohorte mediante la implementación de soluciones computacionales prácticas que evidencien la apropiación y aplicación de los conceptos abordados.Este repositorio contiene las implementaciones correspondientes a las exposiciones seleccionadas, enfocadas en técnicas de clustering y reducción de dimensionalidad.

## Algoritmos Implementados

Este proyecto incluye implementaciones prácticas de:
- DBSCAN (Density-Based Spatial Clustering of Applications with Noise)
- UMAP (Uniform Manifold Approximation and Projection)
- t-SNE (t-Distributed Stochastic Neighbor Embedding)

Cada técnica cuenta con:
- Código funcional
- Experimentación y visualización
- Análisis de resultados
- Documentación técnica asociada

## Guía de Ejecución
1. Crear entorno
python -m venv venv
source venv/bin/activate   # Linux/Mac
venv\Scripts\activate      # Windows

2. Instalar dependencias
pip install -r requirements.txt

3. Ejecutar experimentos

Desde Jupyter Notebook o directamente en consola:

jupyter notebook


Abrir los archivos en notebooks/.
